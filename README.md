# Электронная библиотека

АИС «Электронная библиотека» — веб-приложение на Flask для автоматизации ведения реестра книг и читателей, поиска книг и написания рецензий.

**Группа:** 241-3211  
**Автор:** Минеева Анастасия  
**Вариант:** 2 — Функциональность формирования подборок книг

---

## Функционал

### Основной
- Каталог книг с обложками, жанрами, годом, средним рейтингом и количеством рецензий
- Пагинация на главной странице (10 книг на странице), сортировка по году (новые первые)
- Три роли пользователей: Администратор, Модератор, Пользователь
- Добавление книг с загрузкой обложки (только Администратор)
- Редактирование книг (Администратор и Модератор)
- Удаление книг с подтверждением через модальное окно (только Администратор)
- Написание рецензий с поддержкой Markdown и выбором оценки (0–5)
- Описание книг в формате Markdown с санитайзером Bleach
- MD5-дедупликация загружаемых обложек
- Защита маршрутов по ролям с flash-сообщениями

### Вариант 2 — Подборки книг
- Раздел «Мои подборки» в навигации (только для роли «Пользователь»)
- Создание подборок через модальное окно
- Просмотр списка книг в подборке
- Кнопка «Добавить в подборку» на странице книги (только для роли «Пользователь»)
- Добавление книги в подборку через модальное окно с селектором

## Технологии

- **Backend:** Python 3, Flask, Flask-Login
- **База данных:** MySQL, mysql-connector-python
- **Frontend:** Bootstrap 5, Bootstrap Icons, EasyMDE
- **Прочее:** Bleach (санитайзер), Markdown, python-dotenv, Werkzeug

## Установка и запуск

### 1. Клонируй репозиторий
```bash
git clone <url>
cd library
```

### 2. Создай виртуальное окружение
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Создай файл `.env`
```
SECRET_KEY=your-secret-key-here
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=library
```

### 4. Создай базу данных
```bash
mysql -u root -e "CREATE DATABASE \`library\` CHARACTER SET utf8mb4;"
tail -n +3 schema.sql | mysql -u root library
```
